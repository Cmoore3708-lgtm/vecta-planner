# Vecta Planner v1.0

Proper React/Vercel/Supabase build.

## Included
- Dashboard
- Daily planner
- 7-day planner
- Month view
- New / Unallocated
- Waiting Jobs
- Chris's Tasks
- Alfie / Jordan / Other boards
- Ramp colours
- Utilisation bars
- Job type presets and estimated hours
- Status workflow: Booked → Arrived → In Progress → Work Complete → Ready to Invoice → Archived
- Search
- Print layout
- Supabase schema
- Local demo mode

## Run locally
npm install
npm run dev

## Supabase
Paste `supabase/schema.sql` into Supabase SQL Editor and run it.
